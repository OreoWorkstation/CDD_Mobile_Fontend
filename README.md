# CDD_Mobile_Frontend

猫狗日记 (Cat & Dog Diary) 移动前端项目

> 猫狗日记是一个可以记录宠物日记的平台，并且可以记录宠物的体重生长曲线、消费曲线, 同时还有社区分享、宠物百科功能。
